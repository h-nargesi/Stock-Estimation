«تعاریف پایه»

K	= ثابت ماکسیمم و مینیمم تغییرات نرمال
	= 0.05

p1	= قیمت جاری
p0	= قیمت پایه

ch	= تغییر قیمت جاری نسبت به قیمت پایه
	= (p1 - p0) / p0

«تبدیل تغییر قیمت به تغییر زاویه»

sch	= تغییرات در مقیاس
	= ch / K

θ	= زاویه تغییرات
	= arc-tan(sch)

y	= استاندارد شده زاویه تغییرات
	= θ / (π / 2)
	
«پیدا کردن قیمت جاری پیش‌بینی شده بر اساس مقدار تغییر زاویه پیش‌بینی شده»

ch	= K * tan θ
chʹ	= K * tan θʹ

p1ʹ	= p1 * (chʹ + 1) / (ch + 1)
